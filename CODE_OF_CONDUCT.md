# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful, constructive, and welcoming. Harassment and personal attacks are not tolerated.

To report an issue, open a [GitHub private advisory](https://github.com/nark/WiredSwift/security/advisories/new) or contact the maintainer directly via GitHub.
